# 🎓 Student Result Analyzer (Web UI Version)

An interactive and user-friendly Student Result Analyzer built using C++ (core logic) and redesigned with a modern Web UI using HTML, CSS, and JavaScript.

This project allows users to add students, calculate total marks, average, grade, pass/fail status, and identify the topper.

---

## 🚀 Features

- ➕ Add Student with multiple subject marks
- 📊 Automatic Total & Average Calculation
- 🏅 Grade Assignment (A, B, C, D, F)
- ✅ Pass / ❌ Fail Detection
- 🥇 Topper Identification
- 💾 Persistent Data Storage (LocalStorage / File Handling)
- 🎨 Modern Responsive UI Design
- 🌙 Clean and User-Friendly Interface

1. Clone the repository
2. Open `index.html` in your browser
3. Start adding students 🎉

---

## 🖥 How to Run (C++ CLI Version)

Compile:

```
g++ main.cpp -o result
```

Run:

```
./result add 101 Nandini 90 95 88
./result showall
./result topper
```
